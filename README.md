# Magical Orchestra

Welcome to Lyndsey and Julia's Magical Orchestra!

## Usage

```
lein run -m magical-orchestra.server
```

Overtone takes a long time to start for some reason, so you will have
to do some waiting.

Then go to http://localhost:8080.
